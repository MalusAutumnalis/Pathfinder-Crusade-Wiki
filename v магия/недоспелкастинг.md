# Недоспелкастинг

Недоспелкастеры не имеют PR и не добавляют бонус от него к тестам магии, а используют навык {{Название класса}} Sorcery ({{Характеристика Класса}}) для тестов магии. Для паладинов эта характеристика Willpower, бардов — Fellowship, рейнджеров — Perception.  

Заклинания стоят количество маны, равное уровню таланта, который давал это заклинание.

Недоспелкастер может поддерживать лишь одно заклинание. Поддерживание этого заклинания не увеличивает сложность теста магии для произнесения заклинания.

Для каждого класса тематический набор доступных заклинаний.

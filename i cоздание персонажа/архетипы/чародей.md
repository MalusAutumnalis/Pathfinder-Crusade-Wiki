##### Трейты:  
###### **Spontaneous Spellcasting**   
**Запас маны:** W.b + P.b + PR×2  
Несвязанный заклинатель, использует Willpower для тестов магии.  
Может знать не более W.b + I.b + PR заклинаний. При повышении уровня PR или при проведении соответствующего ритуала может забыть до PR известных заклинаний, вернув потраченный опыт, но обязан немедленно потратить его на изучение других заклинаний.  
Может использовать любые заклинания, которые знает.
